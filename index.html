About

As a full-stack software developer, I specialize in both front-end and back-end development, creating robust and user-friendly web applications. My portfolio website is a testament to my skills and experience in the field.

Front-End Development

In the front-end, I focus on building intuitive and responsive interfaces using modern web technologies. I employ HTML, CSS, and JavaScript, along with frameworks like React and Angular, to craft visually appealing and highly functional user experiences. My approach ensures that the design is not only aesthetically pleasing but also accessible and easy to navigate.

Back-End Development

On the back-end, I leverage my expertise in server-side technologies to develop efficient and scalable applications.
